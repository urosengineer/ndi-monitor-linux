# Changelog

All notable changes to NDI Monitor for Linux will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-03

### Added
- Initial public release
- Network scanning for NDI® sources with automatic discovery
- Real-time video preview with hardware acceleration
- Modern dark mode UI built with Jetpack Compose Desktop
- Single-click source selection and preview
- Bundled Java runtime (self-contained, no Java installation needed)
- Desktop integration with application menu entry
- System-wide command-line access
- Professional installation and uninstallation scripts
- Support for Linux x64 systems

**Note:** Requires NDI® SDK 6.x to be installed separately. Download NDI SDK from https://ndi.video/for-developers/ndi-sdk/

### Technical Details
- Built with Kotlin 2.0.21
- Jetpack Compose Desktop 1.7.3
- Skia-based hardware-accelerated rendering
- NDI SDK 6.x integration via JNI bridge
- Bundled OpenJDK 17 runtime
